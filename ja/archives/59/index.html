はじめに

個人的にFedoraをインストールした後に導入したあれこれだよ。

ターミナルショートカットキーの変更

設定 - キーボード を開いて、一番下の ショートカットキーの表示とカスタマイズ をクリック。

ショートカットキーをカスタマイズする

名前：シェルターミナル

コマンド：gnome-terminal

ショートカットキー：自分で設定してね

システムの更新

1

sudo dnf update

スクリーンショットのショートカットキーを変更

やっぱりWindowsのショートカットキーが慣れてるんだよね。

上記のスクリーンショットのところ、対話的なスクリーンショットをWin+Shift+Sに変更しよう。

ユーザーディレクトリを英語に変更

日本語のディレクトリって、ターミナルで操作するのがすごく大変なんだよね。

まず言語を英語に変更するよ。

1

export LANG=en_US

それからユーザーディレクトリを更新。

1

xdg-user-dirs-gtk-update

この時、ユーザーディレクトリ下のフォルダを英語に変更するかどうか聞かれるから、「はい」を選んでね。

次にシステム言語を日本語に戻すよ。

1

export LANG=zh_CN.UTF-8

もう一度ユーザーディレクトリの更新コマンドを実行。

1

xdg-user-dirs-gtk-update

もう一度上のコマンドを入力すると、日本語に変更するかどうか聞かれるから、「いいえ」を選んで、「今後表示しない」にチェックを入れればOK。

Nvidiaドライバーのインストール

インストール

1

sudo dnf install akmod-nvidia

オプションだよ

1

sudo dnf install xorg-x11-drv-nvidia-cuda

インストールが終わったら再起動してね。

1

reboot

ノート：もし再起動後に「Nvidia kernel module missing falling back to Nouveau」と表示されたら

  1. まずBIOSでセキュアブートが無効になっているか確認してね

  2. rpm -qa akmodsを実行して、インストールされているakmodsパッケージのバージョンをチェック。

  3. sudo akmods --force && sudo dracut --forceを実行して、カーネルモジュールをビルドしてね。

参考にしたのはここ： Fedora 34 NVIDIA kernel module missing. Falling back to nouveau - #4，来自 vk2bea - Ask Fedora

もし上記の方法でダメだったら、以下の方法を試してみてね。

  1. Nvidia関連のものを全部アンインストールするよ。sudo dnf remove \*nvidia\*

  2. ドライバーをインストール。sudo dnf install akmod-nvidia

参考にしたのはここ： Nvidia kernel module missing falling back to Nouveau (Fedora 36 PR) : Fedora
