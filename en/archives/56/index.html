Most mainstream streaming platforms support embedding code for external links. However, as of August 11, 2022, Bilibili’s (when is it shutting down?) default embed link defaults to 360P and doesn’t allow quality adjustments. It’s pretty frustrating compared to YouTube. Below are the parameters and methods for customizing Bilibili’s embed links.

Bilibili Embed Key Reference

KeyDescription
aidThe old AV ID format used by Bilibili
bvidThe current BV ID format
pageVideo index in a series, starting from 1 (default is 1)
as_wideWidescreen mode [1: Widescreen, 0: Standard]
high_qualityHigh quality [1: High quality (up to 1080p) / 0: Lowest quality (default)]
danmakuToggle bullet comments [1: On (default), 0: Off]
vd_source=XXX&t=XXXCan be used for specific timestamp jumping
