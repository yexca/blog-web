VS Code 右下角嘉然

閱讀時間: 3 分鐘

简体中文 English 日本語

前言

首先我不是嘉心糖，我只是覺得很可愛 (我剛開始甚至都不知道是嘉然)，然後這個 Bug 蠻多，自動啟動不是每次都能觸發，但她太可愛啦

然後這個擴充套件還可以改背景，但我個人不是太喜歡就沒改

效果預覽如下

安裝

開啟 VS Code 然後在 擴充套件 搜尋 A-SOUL-live2d 安裝

擴充套件網址： A-SOUL-live2d - Visual Studio Marketplace

其實可以直接去擴充套件那邊看，有安裝過程，不用看我寫的垃圾文章

簡易使用

安裝完畢重新啟動 VS Code，然後在資源檢視最下方會有 LIVE2D-A-SOUL，展開 (不能展開就重新啟動 VS Code)

然後在 補充設定 處 產生 擴充套件依賴檔案，重新啟動 VS Code，點擊 啟動 live2d

右下角就會出現嘉然囉，可以把滑鼠移上去，拖曳右上角第三個圖示以移動位置，第一個點擊改變大小

調整完位置，大小後可以在 設定資訊 處開啟 自動啟動，然後點擊 儲存目前設定 就可以開啟 VS Code 就能看見嘉然囉~

其他

以下複製自擴充套件介紹

  • 基本操作

    • 啟動、關閉 live2d： 字面意思，會啟動看板人物，初始預設位置右下角
    • 儲存目前設定： 在調整 live2d 大小縮放和拖曳位置後，可儲存資訊，下次啟動時自動帶入
    • 重設預設位置： 目前位置異常，無法拖曳移動時可重設使用【縮放大小也會重設】
    • 背景圖片
      • 點擊切換： 點擊按鈕為人物右側圖示第二個
      • 儲存背景圖片： 需要目前背景圖片存在才會生效。只能儲存一份，再次點擊會覆蓋舊的
      • 載入背景圖片： 載入儲存的背景圖片
      • 定時切換： 字面意思，可查看 切換按鈕 是否旋轉判斷是否開啟定時功能
      • 背景圖片樣式設定
      • 下載背景圖片

  • 設定資訊

    • 自動啟動： 字面意思，開啟後。VS Code 啟動，live2d 自動啟動
    • 定位依賴： 人物定位的依賴角

  • 補充設定

    • 擴充套件依賴檔案：
      • 擴充套件依賴檔案會在初次安裝擴充套件並啟動時自動產生
      • 產生： live2d 無法正常啟動時，可嘗試點擊該按鈕，強制重新產生覆蓋設定資訊
      • 移除： 解除安裝該擴充套件前，請盡可能先執行該操作。可移除擴充套件對 VS Code 檔案的所有修改

  • 人物功能

    • 目光跟隨滑鼠 【缺點，暫時無法實現整個頁面的跟隨】
    • 點擊互動
    • A-SOUL 粉絲導航網站入口
    • 切換背景圖片
    • 切換模型
    • 溜冰場
    • 音訊測試
    • 一個魂二創網站入口
    • 模型來源

關於警告

使用時會跳出 Code 損壞警告，原作者指出

本擴充套件是透過修改 VS Code 的 js 檔案的方式執行 所以會在初次安裝，或者 VS Code 升級的時候，出現以下提示，請選擇 【不再提示】

音訊支援

目前擴充套件僅一個簡易的語音測試，暫且可以不用考慮該功能

VS Code 使用的 Electron 版本不包含 ffmpeg ，需替換內建的 ffmpeg 動態連結庫才能正常播放 (每次更新 VS Code 都需重新替換)

VS Code for Windows 1.31.0 - 1.35.1 不需替換，1.36.0 後無此待遇

VS Code for macOS 1.43+ 替換後閃退 解決方案

解除安裝

首先將 補充設定 移除，然後在 擴充套件 介面解除安裝擴充套件即可
