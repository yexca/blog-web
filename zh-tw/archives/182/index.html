引言

最近因為更換硬碟而重新安裝了電腦系統，當我拿出那多年未動的隨身碟時，裡面的軟體多少有點過時，有些已經無法在新系統上執行，還有好多我已經使用了相同功能的替代品，以及使用了一些新的軟體，遂寫下此文章記錄裝機用到的軟體。

回頭發現這是 24.09.16 的文章，中間因為有事耽擱，加上狀態不是特別好，直到 24.10.02 才把文章寫完。

系統

因為用了較長時間的 Win11，所以還是打算繼續使用 Win11。隨身碟裡的系統太舊了，於是去 MSDN itellyou 下載了新版映像檔。

但是在安裝的時候卻發現這玩意居然必須要連網才能安裝了，就是在設定新系統的時候必須連網登入微軟帳戶，要是沒有網路的話最好使用舊版本 (記得應該是可以跳過的) 或者裝 Win10 (然後再升級 Win11 (doge))。

2024.10.03
今天刷到了 兩個跳過網路驗證的方法 ，沒嘗試過，先記錄下來
第一個是按 Ctrl+Shift+F3 可以直接跳過，同時避免建立在地使用者 (下次重新啟動將重新進行設定)
第二個是在連網介面按 Shift+F10 然後輸入 oobe\bypassnro.cmd 之後系統會重新啟動，重啟後在網路頁面會出現沒有網路連接的選項

軟體相關

我的安裝隨身碟已經好久沒有使用了，裡面的軟體有些我已經不再使用，並且也開始使用一些新的軟體，部分替代了原本的功能，所以還是有必要寫一下的。說不定下次裝機看這篇文章的時候我又會換一批軟體了 (雖然現在我不是很喜歡麻煩的事情)。

辦公軟體

基礎的 Office 必不可少，採用 Office Tool 進行安裝方便快速。

以下內容僅安裝 Word、Excel 和 PowerPoint。

下載 Office Tool: https://otp.landian.vip/zh-cn/download.html

啟動 Office Tool，執行程式碼安裝：

1

deploy /add O365ProPlusRetail_zh-cn /O365ProPlusRetail.exclapps Access,Bing,Groove,Lync,OneDrive,OneNote,Outlook,Publisher,Teams /channel Current /dlfirst

利用 kms 啟動 Office，執行程式碼啟動：

1
2
3

ospp /inslicid MondoVolume /sethst:kms.loli.beer /setprt:1688 /act
# or use kms.03k.org
ospp /inslicid MondoVolume /sethst:kms.03k.org /setprt:1688 /act

參考文章

https://www.coolhub.top/archives/11

https://www.coolhub.top/archives/14

下載工具與雲端硬碟

迅雷: 使用的是忘了從哪找的精簡版。雖然會吸血，但有些東西確實還是得用迅雷下載。

BitComet: BT 下載工具，同樣使用了精簡版。類似的工具有 qBittorrent, µTorrent 等。

Resilio Sync : 基於 BT 技術的同步軟體，方便各端檔案同步。

Internet Download Manager: 即 IDM，比較好用的多執行緒下載工具。

115 : 用於下載一些下不動的 BT。類似的工具有 pikpak。

百度網盤 : 開了會員很好用，但沒會員就是折磨。時間，加錢可得。與之類似的如阿里網盤、夸克網盤可以在需要時再下載。

社群軟體

暫時使用的社群軟體也就那樣吧。

Tencent

  • QQ : 最近出的 NT 版本確實還行吧。
  • WeChat: https://www.wechat.com/

Telegram: https://desktop.telegram.org/

解壓縮工具

之前的電腦上倒是裝了不少的解壓縮工具，但實際上使用的也就兩個，於是這次就只裝兩個了。

Bandizip : 應該是較為流行的吧，之前當我剛聽說時學校電腦教室都用上了。

WinRAR: 比較古老的軟體，但有些 rar 只有使用此軟體才能解壓縮。

其他的:

  • 7-zip : 最為流行的解壓縮軟體。
  • NanaZip : 7z 的美化版，並且可以在 Win11 第一層右鍵選單顯示。

其他工具

Utools : 非常便捷的工具，官網有動畫介紹，透過外掛擴充使其非常方便，比如本文就是使用其 MarkDown 擴充寫的，雖然有些地方不如 Typora 但勝在方便。

QuickLook : 快速預覽檔案的工具，用起來不是太理想，希望未來可以找到替代品。非安裝版需要解鎖後使用: https://github.com/QL-Win/QuickLook/issues/1

1Password : 密碼管理工具，可以使用開源替代品 bitwarden 。我已經徹底離不開密碼管理器啦。

PDF 閱讀器: 感覺大部分都差不多，隨緣用用。

  • Foxit PDF
  • Wondershare PDFelement
  • PDF XChange

sandboxie : 沙盒工具，對於不想直接在電腦上執行而污染環境的程式很好用，非安裝版每次執行需要管理員權限。

TickTick : To Do List 軟體，全平台同步，不過最近感覺同步不是太順暢。

QuiteRSS : RSS 訂閱管理工具，但介面較為古老，不過由於我使用習慣了也無所謂。畢竟轉移只需壓縮解壓縮，RSS 訂閱也不會遺失。

Duplicate Cleaner Pro: 之前用的清理軟體找不到了，就用八醬的這個啦。雖然有些時候不是太好用，但使用久了後還行吧。

PixPin : 螢幕截圖工具。

Code Tools

VS Code : 只能說裝機必備了。

Typora : Markdown 視覺化書寫工具。

Git: https://git-scm.com/downloads/win

Git 安裝後需要設定名稱和電子郵件。同時可以更新一下 Github 的 SSH 公鑰了。

1
2

git config --global user.name "Your Name"
git config --global user.email "email@example.com"

參考文章: https://blog.yexca.net/zh-tw/archives/95

JetBrains

  • IntelliJ IDEA : Java 程式設計。
  • GoLand : Go 程式設計。

Dataflare : 連接資料庫的工具。

Docker

我已經完全離不開 Docker 了。

下載地址: https://docs.docker.com/desktop/install/windows-install/

單獨分一個章節是因為安裝時出錯 WSL Update Failed。

其實就是需要去 下載最新版本的 WSL ，然後重啟電腦就好了。

參考文章

https://forums.docker.com/t/wsl-update-failed/140473/3

https://github.com/docker/for-win/issues/13845

https://github.com/docker/for-win/issues/13580#issuecomment-1619667316

MySQL Docker : 因為使用其儲存一些資料，所以還是挺有必要使用的，但使用頻率不是太高，我覺得可以直接裝在 Docker 裡，這樣換系統之類的也方便 (也想過裝在伺服器上，但覺得有時候可能會沒有網路連線)。

安裝

1

docker pull mysql:5.7

執行時需要設定連接埠和 ROOT 密碼的環境變數 MYSQL_ROOT_PASSWORD。

部落格

因為我部落格使用 Jekyll 框架，還需要重新安裝一下，希望不要出問題。

執行環境

  • Ruby: https://www.ruby-lang.org/en/downloads/
  • RubyGems: https://rubygems.org/pages/download
  • Node.js: https://nodejs.org/zh-tw

安裝 Jekyll

1

gem install jekyll

初始化部落格

1

bundle

好吧，有幾個外掛我重啟也裝不上，同時參考了 Installation Issues on MacOSX - Gem::Ext::BuildError: ERROR: Failed to build gem native extension 等文章也無法安裝。

經朋友的提醒，確實還是得用 Docker 比較好，天天搞環境會煩死，明明專案都用 Docker 我部落格居然還在用在地安裝。

參考我 主題安裝文章 安裝 VS Code 外掛 Dev Containers extension 用於查看容器內部檔案。

打開 VS Code 按下 F1 輸入 Dev Containers: Clone Repository in Container Volume... 選擇後輸入一個 Git 連結 (即我的 blog 的原始碼)。

等待一會後 (右下角可以看日誌) 選擇要使用的系統、軟體等配置，我選擇 Debian 系統，軟體我沒選。

首先更新軟體：

1

sudo apt-get update

參考 官方教學 安裝 Jekyll：

1
2
3

sudo apt-get install ruby-full build-essential
# 上方是依賴項目
sudo gem install jekyll bundler

安裝 NodeJS：

1

sudo apt install nodejs

安裝依賴項目：

1

sudo bundle

會提示不要用 root 權限執行，但這只是容器，隨便弄，無視。

嘗試部署專案：

1

bundle exec jekyll s

專案成功執行，但因為是容器的連接埠，需要映射到本機才能看，打開直接能看，VS Code nb (doge)。

感覺比 Win 編譯的要快，非常 Nice。
